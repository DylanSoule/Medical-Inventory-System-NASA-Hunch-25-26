Project Developments

See how H.E.L.I.O.S evolved through design reviews, testing, and feedback from NASA engineers.

Research & Design Documentation

Technology Selection: We selected facial recognition for quick authentication (under 1 second) that functions reliably in microgravity without requiring crew members to carry additional equipment. The SQLite database provides local storage for spacecraft computers

Testing: Comprehensive testing included facial recognition accuracy across lighting conditions and different angles, UI responsiveness on Raspberry Pi hardware, and error handling for different interruptions.

References: NASA HUNCH. "Medical Inventory System Project Specifications." 2025. Link NASA HUNCH. "MCT System Requirements." 2025. Link
  • Sep 2025

    First Facial Recognition Demo

    We successfully implemented the main facial recognition system using InsightFace and OpenCV. This milestone marked our first working prototype that could accurately identify users and log their interactions with medical supplies. The system had real-time face detection with visual feedback, showing a green box around detected faces. This proof-of-concept validated our technology choice and confirmed that facial recognition could work as an authentication method.

  • Oct 2025

    UI & Status Improvements

    We redesigned the user interface to provide better visual feedback and system status information. The main interface was redone with a cleaner layout featuring larger, more readable text and improved button placement for easier navigation. We added a status indicator that shows the current camera state.

  • Oct 2025

    Database Migration: CSV to SQLite

    We transitioned from CSV file storage to a SQLite database system, an important upgrade for data scalability. SQLite's lightweight footprint makes it ideal for spacecraft computers with limited resources while supporting queries for reporting and analysis. This migration enabled features like deletion history, user authentication, and the addition of the actual medications used on the ISS.

  • Nov 2025

    Admin & Deletion History

    Security and data integrity became our focus with the implementation of an admin authentication system using PIN codes. This feature ensures that only authorized personnel can modify inventory records or delete entries. We developed a deletion history feature that maintains a permanent audit trail of all removed items, including who deleted them and when. The system now stores all deletion records in the database, preventing accidental or unauthorized data loss.

  • Dec 2025

    New Facial Recognition & Log

    We created a major overhaul of the facial recognition engine, significantly improving accuracy and speed. The updated system now processes faces in under 1 second with better detection in varied lighting conditions, crucial for spacecraft environments. The scan logging interface received a complete redesign with a more modern, streamlined appearance that displays transaction history in an easy-to-read format. We added real-time visual confirmation when items are scanned, showing item details, quantity changes, and user information instantly.

  • Jan 2026

    Modern UI & Admin Window

    In preparation for our Critical Design Review (CDR) with NASA, we finalized the user interface with a polished, professional appearance. The main UI now features a cohesive design optimized layouts for both desktop and Raspberry Pi displays. We redesigned the admin code entry window with a more secure and user-friendly keypad interface, featuring larger buttons and clearer visual feedback. The entire system was optimized for performance, reducing load times and improving responsiveness to meet NASA's usability standards for space applications.
