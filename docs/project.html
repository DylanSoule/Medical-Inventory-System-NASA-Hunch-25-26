Project Developments

See how H.E.L.I.O.S evolved through design reviews, testing, and feedback from NASA engineers.

Research & Design Documentation

Technology Selection: We are using facial recognition for quick authentication (under 1 second) that functions reliably in microgravity without making crew members carry additional equipment.

Testing: Our testing included facial recognition accuracy across lighting conditions and different angles, UI responsiveness on Raspberry Pi hardware, testing of different python files using pytest, and error handling for different interruptions.

  • Sep 2025

    First Facial Recognition Demo

    We successfully implemented the main facial recognition system using InsightFace and OpenCV. This marked our first working prototype that could accurately identify users and log their interactions with medical supplies. The system had real time face detection with visual feedback, showing a green box around detected faces. This validated our technology choice and confirmed that facial recognition could work as an authentication method.

  • Oct 2025

    UI & Status Improvements

    We redesigned the user interface to give better visual feedback and system status information. The main interface was redone with a cleaner layout featuring larger, more readable text and better improved placement for easier navigation. We added a status indicator that shows the current camera state.

  • Oct 2025

    Admin & Deletion History

    Security and data integrity became our focus with the implementation of an admin authentication system using PIN codes. This feature makes sure that only authorized personnel can modify inventory records or delete logs. We developed a deletion history feature that maintains a permanent trail of all removed items, including who deleted them and when. The system now stores all deletion records in the database, preventing accidental or unauthorized data loss.

  • Nov 2025

    Database Migration: CSV to SQLite

    We transitioned from CSV file storage to a SQLite database system, an important upgrade for data scalability. SQLite makes it ideal for spacecraft computers with limited resources while also supporting queries for reporting and analysis. This change enabled features like deletion history, user authentication, and the addition of the actual medications used on the ISS.

  • Dec 2025

    New Facial Recognition & Log

    We created a major overhaul of the facial recognition engine, significantly improving accuracy and speed. The updated system now processes faces in under 1 second with better detection in varied lighting conditions, which is crucial for spacecraft environments. The scan logging interface received a complete redesign with a more modern, clean appearance that displays transaction history in an easy to read format. We added real-time visual confirmation when items are scanned, showing item details, quantity changes, and user information instantly.

  • Jan 2026

    Modern UI & Admin Window

    In preparation for our Critical Design Review (CDR) with NASA, we finalized the user interface with a polished, professional appearance. The main UI now features a cohesive design optimized layouts for both desktop and smaller tablet displays. We redesigned the admin code entry window with a more secure and user friendly keypad interface, featuring larger buttons and clearer visual feedback. The entire system was optimized for performance, reducing load times and improving responsiveness.
